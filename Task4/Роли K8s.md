| Роль                     | Права роли                                            | Группы пользователей                            |
|--------------------------|-------------------------------------------------------|-------------------------------------------------|
| cluster-platform-admin   | Полный доступ к кластеру                              | Админы Kubernetes                               |
| cluster-security-auditor | Readonly всего кластера                               | Специалист по ИП для аудиста                    |
| ns-client-mart-deployer | CRUD на deployments сервисов Витрины продаж           | Разработчики витрины продаж                     |
| ns-client-mart-qa       | Read-only доступ к сервисам Витрины продаж + логи     | QA витрины продаж                               |
| ns-client-mart-debug    | Просмотр логов и exec в pod'ы Витрины продаж          | Разработчики витрины продаж (debug)             |
| ns-tenant-deployer      | CRUD на deployments сервисов собственников            | Разработчики витрины сервисов для собственников |
| ns-tenant-qa            | Read-only доступ к сервисам собственников + логи      | QA витрины сервисов для собственников           |
| ns-tenant-debug         | Просмотр логов и exec в pod'ы сервисов собственников  | Разработчики сервисов собственников (debug)     |
| ns-accountant-deployer  | CRUD на deployments систем бух учета                  | Разработчики системы бух учета                  |
| ns-accountant-qa        | Read-only доступ к системам бух учета + логи          | QA системы бух учета                            |
| ns-accountant-debug     | Просмотр логов и exec в pod'ы систем бух учета        | Разработчики системы бух учета (debug)          |
| ns-auth-deployer        | CRUD на deployments систем авторизации                | Разработчики систем авторизации                 |
| ns-auth-qa              | Read-only доступ к системам авторизации + логи        | QA систем авторизации                           |
| ns-auth-debug           | Просмотр логов и exec в pod'ы систем авторизации      | Разработчики систем авторизации (debug)         |
| ns-data-deployer        | CRUD на deployments систем хранилища данных           | Разработчики систем хранилища данных            |
| ns-data-qa              | Read-only доступ к системам хранилища данных + логи   | QA систем хранилища данных                      |
| ns-data-debug           | Просмотр логов и exec в pod'ы систем хранилища данных | Разработчики систем хранилища данных (debug)    |
